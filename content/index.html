F# Resources

A curated collection of helpful blogs, videos, and tutorials for aspiring F# developers. These are resources that I have personally found helpful as I learn more about the language and ecosystem. I plan to add more resources as I come across them and dig through my bookmarks. Pull requests are welcome!

I recommend any total F# newcomer to start with fsharpforfunandprofit.com, but there is so much information there that I have added various sections from that site as individual links to this list.
